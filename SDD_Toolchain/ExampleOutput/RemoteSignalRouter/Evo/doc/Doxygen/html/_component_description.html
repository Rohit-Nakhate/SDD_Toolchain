Component Description

General

The module is required since VW has not provided any communication matrix for the internal comunication inside the ECU and the Signals received by ComServFlex via services, which should reach Can networks controlled by Chorus.
The RemoteSignalRouter shall detect these PDUs and add them in its own configuration. These Pdus will be refered as Tunnel Pdus
The generation is performed by calling the build.bat file from RemoteSignalRouter/VARIANT/tools/RemoteSignalRouter_Generate.bat directory.
No parameters are required.
